Introduction

지역별 교육 기회의 불평등성 및 청소년 비행 문제 분석을 위해 지역별 중학생을 대변할 수 있는 데이터를 분석하고, 각 변수 사이의 상관관계 실증하는 것에 이번 프로젝트의 목적이 있다.

분석 결과물을 대쉬보드 나타내고자 했고, 개인적으로 도전해보고 싶었던 것은 무료 도메인을 활용해 대쉬보드를 배포해 보는 것이었다. Plotly의 Dash를 주로 활용했으며, 최종 대쉬보드는 이곳에서 확인할 수 있다.
무료 호스팅 서버가 구동될 때 까지 약 15초 정도 시간이 소요된다

Method


Data

- 2018-2022 지역별 중학교 졸업 후 상황
- 2018-2022 지역별 중학교 사교육비 지출
- 2018-2022 지역별 청소년 음주율, 흡연율 (청소년 비행)
- 2018-2022 지역별 청소년 우울감 경험률

Visualization

데이터의 지역별 시각화는 지오서비스웹(GEOSERVICE-WEB)이라는 곳에서 제공하는 행정구역도 데이터를 JSON 파일로 변환해 사용했다. JSON 파일 내 ‘CTP_KOR_NM’을 수집한 데이터의 지역명과 일치시켜 줌으로써 위치 데이터를 받아올 수 있었다.

최종적으로 Plotly 의 Dash 와 합치기 위해 html 로 변환했고 같은 경로 내 assets 폴더를 생성 후 코드 실행 시 참조될 수 있게 했다.

데이터 분포를 지역별로 표시하여 비교하고자, Plotly의 graph_obj를 활용해 Boxplot 을 추가했다.

Hosting

Ploty의 공식 youtube 채널을 참고했고, render 라는 사이트를 활용했다. render 사이트의 호스팅 구조는 필요한 라이브러리를 명시한 requirements.txt 파일과 .py 파일을 git repository에 업로드해 놓으면 되는 간단한 방식이었다. 여러 번의 배포 실패와 이번 프로젝트에서는 총 두 번의 배포를 성공했다. render의 무료서비스는 최대 500 분의 build 제한 시간이 있지만 처음 사용해 보기에 충분한 듯 보인다.
